# Real Estate Property Listing Project (Laravel + MySQL + Bootstrap)

Proyecto inmobiliario **educacional** para construir un sistema de **listado y búsqueda de propiedades** (venta/renta), con **dashboard de usuario** y **panel de administrador**.

Educational real estate project to build a **property listing and search** platform (buy/rent), with a **user dashboard** and an **admin panel**.

## Tech stack

- PHP + Laravel
- MySQL
- Bootstrap

## Estado actual

Por ahora solo se ha implementado:

- Instalación base de Laravel (estructura inicial y autenticación scaffold/rutas)
- Assets públicos del curso integrados en `public/assets`

Las funcionalidades del MVP (búsqueda de propiedades, dashboard completo, panel admin) están planificadas y se irán implementando.

## Propósito

Este repositorio es con fines **educativos** y de práctica, para materializar progresivamente un proyecto real.
